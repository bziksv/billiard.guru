Формат **швейцарская 3** (27 матчей, 7 колонок TS).

## SVG

| kind | from → to | Пример |
|------|-----------|--------|
| win, bye | R1 → верхняя R1 (`0 → 1`) | #1+#2 → #9 (вилка) |
| loss | R1 → нижняя R1 (`0 → -1`) | #1+#2 → #13 (вилка) |
| win | соседние колонки R2→R3 | **#9→#21** … **#16→#20** (короткая H–V–H) |
| win | 1/4→полуфинал (`2→3`) | **#21+#22→#25**, **#23+#24→#26** (вилка) |
| win | полуфинал→финал (`3→4`) | **#25+#26→#27** (вилка) |

**Остальное без SVG** — в подвале карточки:

- `победитель на #…`
- `проигравший на #…`

(нижняя R1→крест, верхняя→1/4, крест→1/4, полуфинал, финал).
