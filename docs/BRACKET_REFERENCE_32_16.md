# Эталон турнирной сетки: 32→16 (Fixed Swiss)

**Статус: закреплён.** Реализация эталонной сетки для 32 участников. Менять только по явному запросу; регрессии — через `test-fixed-swiss-layout.ts` и чеклист ниже.

Связанный эталон меньшего масштаба: [`BRACKET_REFERENCE_16_8.md`](./BRACKET_REFERENCE_16_8.md) (паттерны UI, карточки, модалка — те же).

## Название формата

- **Код:** `FIXED_SWISS_32` / `FIXED_PAIR_SWISS_32`
- **Подпись в UI:** `FIXED_SWISS_32_FORMAT_LABEL` в `apps/web/src/lib/validators.ts`
- **С бронзой:** `FIXED_SWISS_32_BRONZE` — **60** встреч (#60 = матч за 3–4 **под #59** в колонке «Финал»)

## Параметры сетки

| Параметр | Значение |
|----------|----------|
| Участников | 32 (31 → bye в 1-м туре) |
| Встреч | **59** (#1–#59), с бронзой **60** |
| Туров в БД | **7** |
| Колонок на экране | **10** (−4 … +5) |
| Олимпийка | с **1/8** (#41–#44), 1/4 с **#53** |

**Устаревшие варианты в БД (не эталон):** 55/56, 63/64 встреч — баннер **«Сформировать заново»**.

### 1/8 + тур 4 → 1/4 (`fixedSwissTs32OlympicToQuarterTarget`)

Каждый 1/4 получает **два входа**: победитель 1/8 (team1) и победитель нижней тур 4 (team2).

| Источник | 1/4 | team |
|----------|-----|------|
| #41 | #53 | 1 |
| #51 | #53 | 2 |
| #42 | #54 | 1 |
| #52 | #54 | 2 |
| #43 | #55 | 1 |
| #49 | #55 | 2 |
| #44 | #56 | 1 |
| #50 | #56 | 2 |

**Y:** #53 на одной линии с #41, #54 с #42, #55 с #43, #56 с #44 — короткая SVG-линия 1/8→1/4 (col 2→3).

**Порядок 1/4 слева направо по Y:** #53, #54, #55, #56 (не 53-55-54-56).

## Вариант с бронзой (`FIXED_SWISS_32_BRONZE`)

| Параметр | Значение |
|----------|----------|
| Встреч | **60** (#1–#60) |
| Отличие от `FIXED_SWISS_32` | +#60 — проигравшие полуфинала (#57, #58) |
| Layout | `buildTsPositions32Bronze` — #60 **под #59**, та же col **5** («Финал») |
| Подвал #60 | «матч за 3–4 место» |
| Подвал #57–#58 | «полуфинал» (не «место 3–4») |

Links (дополнительно к 59-встречному шаблону):

```text
#57 проигравший → #60 team1
#58 проигравший → #60 team2
```

Парный аналог: `FIXED_PAIR_SWISS_32_BRONZE`.
