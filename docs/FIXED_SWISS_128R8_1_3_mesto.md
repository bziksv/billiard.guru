# Эталон: 128→64 R8 elim + бронза (`FIXED_SWISS_128R8_1_3_mesto`)

**Статус:** закреплён. Масштаб [`BRACKET_REFERENCE_64_32.md`](./BRACKET_REFERENCE_64_32.md) для 128 участников с вылетом с 1/8 и колонками TS −4/−5/−6.

## Название формата

| | |
|---|---|
| **Код Prisma** | `FIXED_SWISS_128R8_1_3_mesto` |
| **Подпись UI** | `FIXED_SWISS_128R8_1_3_mesto_FORMAT_LABEL` в `validators.ts` |
| **Без бронзы (два 3-их)** | `FIXED_SWISS_128R8_2_3_mesta` — 247 встреч, та же сетка без #248 |
| **Legacy** | 215/216, 219/220 — старые подсчёты до колонок −4/−5/−6 |

## Параметры

| Параметр | Значение |
|----------|----------|
| Участников | 128 (127 → bye в 1-м туре) |
| Встреч | **248** (#1–#247 финал, #248 бронза) |
| Без бронзы | **247** |
| Туров в БД | **7** |
| half1 / half2 / gridSize | 32 / 64 / 128 |
| Слотов R3 | **84** (= 64 + 4 − 16 + 4 + 28 display) |
| Колонок UI | **15** (−6 … +7) |
| Олимпийка | 1/8 **#233–#240**, 1/4 **#241–#244**, полуфинал **#245–#246**, финал **#247**, бронза **#248** |

## Связи (links) — ключевая логика

### R1 → R2

Стандарт TS: win → верхняя R2, loss → нижняя R2 (`slotParityTeam`).

### R2 → R3 (верхняя тур 2, k=13…16)

Дополнительные win-links для пар R2 (слоты 65–68 и др.) → R3 слоты верхней тур 2.

### Верхняя тур 2 (#161–#176, R3 слоты 41–52, 65–68)

| kind | Куда |
|------|------|
| **win** | R5 #209–#216 (пары `FIXED_SWISS_TS128R8_UPPER_TOUR2_WIN_PAIRS`) |
| **loss** | R3 display нижней тур 4 (#193–#208) |

Проигравший верхней тур 2 попадает в **team1** display-слота нижней тур 4; `fixedSwissTs128R8UpperTour2LossDisplaySlot()`.

### Нижняя тур 3 (#177–#192, R4:1–16)

| kind | Куда | toTeam |
|------|------|--------|
| **win** | R3 display нижней тур 4 (#193–#208) | `fixedSwissTs128R8LowerTour3WinToTeam()` — complement к loss верхней тур 2 |
| **loss** | места 33–48 (не link, placement) | — |

`fixedSwissTs128R8LowerTour3WinTarget(fromSlot)`: R4 slot → R3 display slot через `257 - matchNo`.

### Нижняя тур 4 (#193–#208)

| kind | Куда |
|------|------|
| **win** | R3:69–72 или R5:9–12 (#217–#224) парами |
| **loss** | места 33–48 |

`fixedSwissTs128R8LowerTour4WinTarget`: пары #193+#194→#224 … #207+#208→#217.

### Нижняя тур 5 (#217–#224)

Источники (R3 или R5):

| # | fromRound:slot |
|---|----------------|
| #217 | R3:69 |
| #218 | R3:70 |
| #219–#222 | R5:9–12 |
| #223 | R3:71 |
| #224 | R3:72 |

**win** → нижняя тур 6 (#225–#232), `fixedSwissTs128R8LowerTour5WinToTeam()` — complement к loss верхней тур 3.

### Верхняя тур 3 (#209–#216, R5:1–8)

| kind | Куда |
|------|------|
| **win** | 1/8 #233–#240 (R3:33–40) |
| **loss** | нижняя тур 6 #229–#232 / #225–#228 |

### Нижняя тур 6 (#225–#232)

**win** → 1/8 (#233–#240) по формуле `465 - fromMatchNo`, `fixedSwissTs128R8LowerTour6WinToTeam()`.

### 1/8 → 1/4 → полуфинал → финал

| Этап | Пары R3/R5 слотов | win → |
|------|---------------------|-------|
| 1/8 | `FIXED_SWISS_TS128R8_EIGHTH_WIN_PAIRS`: [33,34]…[39,40] | R5:13–16 (#241–#244) |
| 1/4 | R5:13–16 | R6:1–2 (#245–#246) |
| полуфинал | R6:1–2 | R7:1 (#247) |
| loss полуфинала | R6:1–2 | R7:2 (#248) — только `1_3_mesto` |

### Проигравшие 1/8

Links **loss** с 1/8 **удалены** (фильтр в `buildFixedSwissTsR8ElimForHalf2`). Игроки получают места 33–48 через placement, не через нижнюю тур 4 классического TS.

## SVG / footer-only edges

Линии **не рисуются** (только подпись «победитель на #…») для:

- нижняя тур 4 → нижняя тур 5 (col −4 → −5)
- нижняя тур 5 → нижняя тур 6 (col −5 → −6)
- нижняя тур 6 → 1/8 (col −6 → 4) — **auto-advance включён** (не footer-only с 2025-fix)
- часть нижней тур 3 → нижней тур 4

Проверка: `isFixedSwissWinLinkFooterOnly`, `isFixedSwissLowerTour4FooterWinEdge` в `fixed-swiss-layout.ts`.

## Нумерация `fixedSwissTs128R8ElimMatchNo(round, slot, withBronze)`

| Round | Правило |
|-------|---------|
| R1 | `slot` |
| R2 | lower: `64+slot`; upper: `slot+64` |
| R3 | см. ветки по slot (display / 1/8 / upper tour2) |
| R4 | `193 - slot` |
| R5 | tour3: `208+slot`; tour5 mid: `210+slot`; 1/4: `228+slot` |
| R6 | `244 + slot` |
| R7 | slot 1 → #247; slot 2 → #248 (если withBronze) |

## Размещение (places)

| # / этап | Место |
|----------|-------|
| #247 | 1 (победитель) |
| #248 | 3–4 (бронза) или 4 |
| #247 без бронзы (`2_3_mesta`) | 1–2 |
| полуфинал без бронзы | оба 3-е |
| проигравшие 1/8 | 33–48 |
| нижняя тур 4 loss | 33–48 |

## Отличие от полной TS 128→64 (`FIXED_SWISS_64` / 231–232)

| | Полная TS | R8 elim |
|---|-----------|---------|
| Встреч | 231/232 | 247/248 |
| Нижняя тур 4 в R3 (#97–#112 аналог) | есть, ведёт в 1/8 | **убрана** — вылет с 1/8 |
| Колонки −4/−5/−6 | нет | **есть** (#193–232) |
| 1/8 | #105–#112 (64) | #233–#240 (128) |

| Параметр | 128R8 | 256 R16 |
|----------|-------|---------|
| gridSize | 128 | 256 |
| half1 / half2 | 32 / 64 | 64 / 128 |
| Встреч | 247 / 248 | 455 / 456 |
| R1 | win→#97, loss→#65 | win→#129, loss→#193 |
| Нижних туров | 6 | **5** |
| Oлимпийka | 1/8 #233–#240 | 1/16 (колонка после верхней тур 5) |
| Финал / бронза | #247 / #248 | #455 / #456 |

**Модули:** `fixed-swiss-ts-256r8-grid.ts`, layout — `buildTsPositions256R16()`.
