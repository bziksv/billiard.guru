## Вилки vs короткие линии

- **Вилка** (`gridFixedForkConnectorPath`): только когда **два** источника в **соседней** колонке сливаются в одну цель (R1, крест→#48, верхняя R2→1/8, 1/4→полуфинал).
- **Короткая** (`gridFixedConnectorPath`): соседние col, один источник (#17→крест, #48→#52, #41→#53).
- **Запрещено** для 32→16: `isFixedSwissQuarterSemiForkEdge(3→5)` — не использовать «шину» 1/8+tour4 в один 1/4.

## Y и trunk

- Точка линии: `teamDividerY`, не центр строки.
- Крест #40…#33: **та же Y**, что нижняя R2 #17…#24 (слот в слот).
- Нижняя тур 3 #48…#45: середина между парой креста (#40+#39 → #48 …).
- Нижняя тур 4 #52…#49: середина между той же парой креста (col −4).
- 1/8 #41…#44: середина пары верхней R2; 1/4 #53…#56: **Y = Y своего 1/8**.
