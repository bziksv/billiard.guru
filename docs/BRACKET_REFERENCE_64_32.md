# Эталон турнирной сетки: 64→32 (Fixed Swiss)

**Статус: закреплён.** Масштаб эталона [`BRACKET_REFERENCE_32_16.md`](./BRACKET_REFERENCE_32_16.md) для 64 участников.

## Название формата

- **Код:** `FIXED_SWISS_64` / `FIXED_PAIR_SWISS_64`
- **Подпись в UI:** `FIXED_SWISS_64_FORMAT_LABEL` в `apps/web/src/lib/validators.ts`
- **С бронзой:** `FIXED_SWISS_64_BRONZE` — **112** встреч (#112 = матч за 3–4 **под #111**)

## Параметры сетки

| Параметр | Значение |
|----------|----------|
| Участников | 64 (63 → bye в 1-м туре) |
| Встреч | **111** (#1–#111), с бронзой **112** |
| Туров в БД | **7** |
| Колонок на экране | **10** (−4 … +5) |
| Олимпийка | с **1/8** (#81–#88), 1/4 с **#105** |

## Колонки и нумерация (#)

| Колонка | Заголовок | Диапазон # |
|---------|-----------|------------|
| 0 | Первый тур | #1–#32 |
| −1 | Нижняя, тур 1 | #33–#48 |
| 1 | Верхняя, тур 1 | #49–#64 |
| −2 | Нижняя, тур 2 (крест) | #80–#65 |
| 2 | 1/8 финала | #81–#88 |
| −3 | Нижняя, тур 3 | #96–#89 |
| −4 | Нижняя, тур 4 | #104–#97 |
| 3 | 1/4 финала | #105–#108 |
| 4 | Полуфинал | #109–#110 |
| 5 | Финал | #111 (+ #112 бронза) |
